# COP4710 Project

## Setup for Development
_Requires Docker_

### Using New Docker Compose Syntax (Plugin):
Spin up: 
```shell
docker compose up -d --build
```
Take down:
```shell
docker compose down -v
```

### Using Traditional Docker Compose Syntax (Legacy):
Spin up:
```shell
docker compose up -d --build
```
Take down:
```shell
docker compose down -v
```
